# General Purpose Agents

This directory contains general-purpose agents that can be used across a wide variety of contexts and tasks.

## Available Agents

General purpose agents are versatile and can handle multiple types of tasks. They are ideal starting points when you're not sure which specialized agent to use.

## When to Use

- Multi-faceted tasks requiring diverse capabilities
- Exploratory work where requirements aren't fully defined
- Quick prototyping and experimentation
- Learning and educational purposes

## Creating General Purpose Agents

General purpose agents should:

- Have broad capabilities
- Be well-documented with diverse examples
- Handle graceful degradation for edge cases
- Provide clear limitations

See the [Agent Creation Guide](../../docs/creating-agents.md) for details.
