# Instructions

Detailed instructions for various tasks and workflows.

## Structure

```text
instructions/
├── development/        # Development workflow instructions
├── deployment/         # Deployment and release instructions
├── testing/            # Testing strategies and instructions
└── maintenance/        # Maintenance and update procedures
```

## Format

Instructions should be:

- Clear and actionable
- Step-by-step when appropriate
- Include prerequisites
- Provide examples
- Link to relevant agents or prompts
